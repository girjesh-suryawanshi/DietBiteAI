Prompt for Replit AI: Build FitBite – AI-Powered Global Diet Planner App (Web)
🎯 GOAL:

Build a full-stack, web + mobile-compatible application called FitBite — an AI-personalized global diet planner that helps users achieve Weight Loss, Weight Gain, or Health Maintenance through culturally relevant 7-day meal plans. Users should be able to input their preferences, generate plans using AI, and download/share them in PDF format. App should support progress tracking and be scalable.

💻 TECHNOLOGY STACK (Web-first MVP):
Frontend: React.js with Tailwind CSS

Backend: Node.js + Express.js (Firebase Functions optional)

Database & Auth: Firebase Authentication + Firebase Firestore

File Storage: Firebase Storage (for PDFs)

Notifications: Firebase Cloud Messaging (future phase)

PDF Generation: Firebase Cloud Functions using Puppeteer or pdf-lib

Optional Charts: Recharts
